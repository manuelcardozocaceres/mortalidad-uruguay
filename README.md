## Análisis de Mortalidad en Uruguay (1875-2023)

Análisis descriptivo de defunciones en Uruguay usando datos públicos 
del INE y el MSP. Incluye evolución histórica 1875-2023 y desglose 
por causa y sexo para 2022-2023.

**Herramientas:** R, tidyverse, ggplot2, readxl, RMarkdown

**Fuentes:** Instituto Nacional de Estadística (INE) y 
Ministerio de Salud Pública (MSP) de Uruguay.
